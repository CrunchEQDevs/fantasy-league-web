# Fantasy League App

Aplicação de Fantasy League completa com Flutter e PHP.

## Tecnologias

### Frontend
- Flutter (Dart)
- Provider para State Management

### Backend
- PHP 8.4
- API REST
- MySQL 9.5
- Composer

### Autenticação
- WhatsApp Business API / Twilio

## Estrutura do Projeto

```
fantasy-league/
├── backend/           # API PHP
│   ├── api/          # Endpoints da API
│   ├── config/       # Configurações (DB, etc)
│   ├── models/       # Models do banco
│   ├── controllers/  # Lógica de negócio
│   ├── middlewares/  # Auth, CORS, etc
│   └── utils/        # Funções auxiliares
├── frontend/         # App Flutter
├── database/         # Scripts SQL
└── docs/            # Documentação

```

## Instalação

### Pré-requisitos
- Flutter 3.35+
- PHP 8.4+
- MySQL 9.5+
- Composer

### Setup

1. Clone o repositório
2. Configure o backend (ver backend/README.md)
3. Configure o frontend (ver frontend/README.md)
4. Execute os scripts do banco (ver database/README.md)
